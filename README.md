# FIIMS Fase 1 - Sistema de Gestión de Infraestructura Física

Este repositorio contiene el código fuente para la Fase 1 del proyecto FIIMS, enfocado en el alta de nodos de cableado estructurado en campo.

## Stack Tecnológico

- **Backend:** Node.js, Express, tRPC, TypeScript
- **Frontend Web:** React, Vite, Tailwind CSS, TypeScript
- **Aplicación Móvil:** Android Nativo (Kotlin)
- **Base de Datos:** PostgreSQL
- **ORM:** Drizzle
- **Infraestructura:** Docker, Nginx

## Estructura del Proyecto

- `client/`: Frontend web (React)
- `server/`: Backend (Node.js + tRPC)
- `drizzle/`: Esquema y migraciones de base de datos
- `android/`: Aplicación móvil (Kotlin)
- `nginx/`: Configuración de Nginx
- `shared/`: Código compartido (tipos, validadores)
